1. Изисквания:

Frontend:
-Използвайте HTML, CSS и JavaScript (или фреймуърк като React, Vue.js или Angular).
-Интерфейсът трябва да позволява на потребителите да добавят нови записки, да виждат списък с всички записки, да могат да редактират и да премахват записки.

Backend:
-Използвайте Node.js (или друг сървърен език, който предпочитате, като Python или Ruby).
-Включете базова сървърна логика за съхранение на записки (може да използвате JSON файл или база данни като MongoDB, MySQL и т.н.).
-Реализирайте API за комуникация с фронтенда (например, RESTful API с POST, GET, PUT, DELETE методи).

Допълнителни изисквания:
-Приложението трябва да бъде работещо на локален сървър.
-Кодът трябва да бъде структуриран по най-добрите практики за разработка (например, модулност, използване на контролери, подходящи имена на файлове и т.н.).

2. Указания за изпълнение:
-Можете да използвате всякакви технологии, които познавате включително и ChatGPT
-Използвайте Git за управление на версиите и споделете линк към вашето репо (например, GitHub, GitLab или Bitbucket).
-Включете кратко README с инструкции как да стартирате проекта на локален сървър.
